Proyecto correspondiente al final de la materia Programación Web II de la materia Analista de Sistemas del Instituto Da Vinci
Es una página/tienda relacionada a la marca Nikon, que vende productos relacionados a la fotografía.

Este proyecto está realizado con PHP y JavaScript. También para los estilos se llama por CDN a Bootstrap.
Para el desarrollo se utilizo Xampp, haciendo uso de Apache y PHPMyAdmin.

Cuenta con un login y un registro, funciones de administrador para agregar, actualizar o eliminar productos y usuarios
El proyecto utiliza una base de datos, que se incluye aquí.
Funciones, secciones y páginas están separados en carpetas distintas para mejor organización
